# Radi Online Interpreter

Для того, чтобы запустить ide на своем браузре потребуется лишь запустить сервер и само окно с нашим ide.  

## Подключение сервера
- Для запуска сервера потребуется версия `Golang 1.21.5`. Установите, если ее нет и прейдите к следующему пункту. 
- Перейдите в папку [backend](/backend/) и пропишите следующую команду:  
`go run main.go`  
Если вы сделали все правильно, то увидите `Server is running on port 8080...` и бесконечно запущенную программу. Ваш сервер запущен!  
(если появились проблемы с запуском сервера, возможно проблема в вашем localhost'е, обратитесь к разработчикм)

## Работа с ide
Откройте папку [frontend](/frontend/) и выберете файл [index.html](/frontend/index.html) - это и есть 'экзешник' нашей ide.

После открытия, вы должны увидеть это:
![Ide](https://github.com/MAILabs-Edu-2024/fp-compiler-lab-radiocative-pepelulka/blob/main/ide/frontend/img/ide1.png)
Вас сразу встречает пример програмы на языке *Radi* - вычисления факторила.

Принцип работы в ide очень прост: 
- Вы пишите программу в поле код-редактора
- Если вы хотите ввести сами какие-то числа в программу в поле *Console* пишите свой ввод
- Затем нажимаете в правом верхнем углу нажимаете на треугольник (Кнопка Play) и ожидаете результат в поле *Console*

### Важно!
После того, как вы уже получили результат и хотите получить новый (а в программе требуется ввод), обязательно *СРАЗУ* напишите входные данные перед запуском.
